# **Project Objective**

The Early Warning System for Business Failure Detection project aimed to predict whether a company is likely to fail using Machine Learning techniques and financial indicators.

The project focused on analyzing historical financial and operational data to identify high-risk companies and support proactive business decision-making.

---

# **Dataset**

- Financial and operational business data
- Revenue growth and profit margin details
- Debt-to-equity and cash flow indicators
- Company size, industry, and ownership information
- 50,000 business records with numerical and categorical features

---

# **Key Business Questions**

- Which financial indicators contribute most to business failure?
- How does debt-to-equity impact failure risk?
- Which Machine Learning model performs best?
- Can predictive analytics identify high-risk companies early?
- How do profitability and cash flow affect survival probability?

---

# **Process & Methodology**

### **Data Preprocessing**
- Handled missing values using imputation
- Encoded categorical variables
- Applied feature scaling and outlier treatment
- Split data into training and testing datasets

### **Exploratory Data Analysis (EDA)**
- Analyzed feature relationships using heatmaps and boxplots
- Identified patterns, trends, and correlations
- Evaluated business failure distribution

### **Model Building**
Implemented and evaluated multiple Machine Learning models:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Support Vector Machine (SVC)
- Naïve Bayes
- Gradient Boosting
- XGBoost
- AdaBoost

### **Model Evaluation**
- Compared models using training and testing accuracy
- Applied GridSearchCV for hyperparameter tuning
- Selected Gradient Boosting as the best-performing model

---

# **Key Insights**

- High debt-to-equity increases business failure risk
- Strong profitability reduces failure probability
- Stable cash flow improves business survival chances
- Machine Learning improves financial risk prediction accuracy
